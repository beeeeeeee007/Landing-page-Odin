# Landing Page — The Odin Project

My solution to [The Odin Project](https://www.theodinproject.com/lessons/foundations-landing-page)'s 
**Landing Page** assignment from the Foundations course.

## Goal
Build a responsive landing page using:
- Semantic HTML
- Basic CSS
- Flexbox for layout

The page includes a header/nav, a hero section, an info section with 
multiple call-to-action buttons, and a footer.


## 🛠️ Built With
- HTML5
- CSS3 (Flexbox)

- Structuring a page semantically before styling it
- Using Flexbox for nav bars and button layouts instead of floats
